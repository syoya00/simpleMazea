・操作

simpleMazea
w:上に移動
s:下に移動
a:左に移動
d:右に移動
r:リセット（ステージの再作成）
c:答えの表示
p:ステージの壁数を減らす
l:ステージの壁数を増やす
m:デバッグメニュー（ステージの壁数設定の表示）
左上から右下へと迷路を抜けましょう。
黄色い点があなたです。
cキーを押すとスタート地点からゴールまでのルートが表示されます。
